# applemaya (Maya Flor Portfolio)

작가이자 에코 플로리스트 문혜정(Maya)의 포트폴리오 웹사이트입니다.

## 기술 스택

- **Next.js** (Pages Router)
- **Tailwind CSS**
- **next-i18next** (국/영문 다국어 지원)
- **Framer Motion** (애니메이션)
- **Sharp** (이미지 최적화)

## 주요 구조

- `src/components/`: 레이아웃 및 기능별 컴포넌트
- `src/pages/`: 라우팅 및 페이지 컴포넌트
- `public/locales/`: 번역 데이터 (ko, en)

## 시작하기

```bash
npm install
npm run dev
```
